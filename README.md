## 🔍 Overview
This project provides a comprehensive technological solution for an NGO with a decade of experience promoting young women's public and political leadership. 

1.  **Graduate Management System (Critical Priority):** A centralized internal platform to replace manual Excel tracking, enabling long-term tracking and secure networking.
2.  **Informational Website (Secondary Priority):** A digital landing page to feature the new "Equal Leadership" program.

### **Main Features:**
* 🔍 **Graduate Search Engine:** Secure directory for alumni networking.
* 🏷️ **Peer-Matching:** Filter graduates by specialties and interest tags.
* 💬 **Safe Communication:** WhatsApp integration with approval-based sharing.
* 📊 **Admin Tracking:** Cohort and engagement tracking for fundraising.
